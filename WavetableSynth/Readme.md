## 기능

- Preset 지원
- Voice 설정 가능 
- 2개의 Oscillator
- Wave table 선택 가능
    - Sin, Saw, Tri, Squ
- 각각의 오실레이터 비율 조정
    - Massive 의 컨셉

- Ring Modulation, Amplitude Modulation 지원

- Filter 지원

  - High pass, Low pass, Band pass, Notch Pass

- Delay 지원

  - Dry/Wet값 조절
